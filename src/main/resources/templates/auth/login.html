Hướng dẫn Google OAuth2

Để sử dụng đăng nhập Google, cần cấu hình OAuth2 Client:

1
Truy cập Google Cloud Console

https://console.cloud.google.com/

2
Tạo Project hoặc chọn Project có sẵn

Tạo credentials OAuth 2.0 Client ID

3
Cấu hình OAuth Consent Screen

Thêm email và domain ứng dụng

4
Thêm thông tin vào application.properties
spring.security.oauth2.client.registration.google.client-id=YOUR_CLIENT_ID spring.security.oauth2.client.registration.google.client-secret=YOUR_CLIENT_SECRET
